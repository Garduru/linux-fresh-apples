# Changelog

All notable changes to this project will be documented in this file.

This project loosely follows Keep a Changelog and Semantic Versioning (SemVer).

---

## [1.0.0] - 2026-01-19

### Added
- Initial public release
- Arch / CachyOS focused post-install setup script
- Pacman database updates
- Core CLI and system package installation
- Flatpak and Flathub setup
- Desktop applications installed via Flatpak to avoid duplicates
- Firewalld configuration for KDE Connect
- Optional cleanup to remove duplicate pacman GUI applications
- ProtonPlus installed via Flatpak
- ProtonUp-Qt removed to prevent duplication
- Safety confirmation prompt before any system changes are made

### Notes
- Designed for personal daily-driver setups
- Intended to be simple, readable, and safe
